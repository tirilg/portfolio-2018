A N I M A T I O N


Animation was completely new to me, and very fun and interesting to learn more about. I firstly learned about the work you have to do before you even start animating, such as writing a story, designing characters and creating a storyboard. I was later introduced to JavaScript and all the amazing animations you can do through code. In the final project of the theme my group and I had to create an animated teaser for an old movie called "The Big Sleep", and through this project I got a lot more familiar with illustrator, and got an introduction to After Effects.

The Big Sleep.

Interactive teaser we showed to a group of students as an exhibition. I did several of the illustrations as well as some work in After Effects.

Simple animation.

A simple animation of different movements coded in HTML, CSS and Javascript.

Interactive animation.

In this group project, my character was used for an interactive animation coded in HTML, CSS and JavaScript. I did all the illustrations and the storyboard.
